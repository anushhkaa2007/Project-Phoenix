# 🚀 Project Phoenix

> **Build yourself while building your software.**

## About

Project Phoenix is a personal productivity dashboard that I am building over 15 days to learn software development while improving my discipline, consistency, and problem-solving skills.

The goal is not simply to finish a web application, but to experience the complete software development process—from planning and designing to building, debugging, documenting, and deploying.

This project represents my journey as a first-year B.Tech student learning to think like a software developer.

---

## Goals

- Learn HTML, CSS and JavaScript
- Practice Git and GitHub
- Build a real-world web application
- Develop consistency through daily coding
- Learn to use AI as a coding partner
- Build a portfolio-worthy project
